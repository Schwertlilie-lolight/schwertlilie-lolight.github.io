2. 与类型参数 T 的不同

类型参数 T 和 通配符 ? 似乎都是用于实现泛型灵活性的工具；

但是，实际上它们有着很大的不同

2.1 使用位置不同

在泛型中，类型参数 T 是一个 确定的 类型，

通常用于泛型类和泛型方法的 定义，不能用于调用代码。

而通配符是一个 不确定 的类型，通常用于泛型方法的 调用代码 和形参，不能用于定义类和泛型方法

2.2 类型参数保证形参一致性

基于 T 是一个确定的类型，那么就可以通过 T 来 确保 泛型参数的一致性；
