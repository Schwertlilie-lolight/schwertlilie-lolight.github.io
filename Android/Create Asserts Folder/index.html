Create Asserts Folder

发表于 2016-10-18 | 更新于: 2018-01-13 | 分类于 Android |
字数统计: 142 | 阅读时长 ≈ 0:01

概述

Android 如何创建 Asserts 文件夹，用来存储一些需要内置在 app 里的大资源呢？

步骤如下：

1. 右边栏移动到 Android 视图

2. 在 app 处右键

New -> Folder -> Asserts Folder

3. 按照对话框提示进行即可
