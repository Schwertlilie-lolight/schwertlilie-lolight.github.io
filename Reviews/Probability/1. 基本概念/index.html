1.1 随机试验

  1. 可重复
  2. 结果不唯一
  3. 结果不确定

随机性：包括上述性质以外还有

  1. 结果范围确定
  2. 试验完成，必有一个确定结果

1.2 样本空间

样本空间是随机试验的所有可能结果集合

1.3 随机事件

样本空间的子集叫做随机事件，元素称为样本点

基本事件：单点集合
必然事件：样本空间本身，记为 Ω\OmegaΩ
不可能事件：空集 ∅\varnothing∅

1.5.2 概率

1.5.2.1 统计定义

在一组相同条件下，重复地做 n 次试验，记, nAn_An​A​​为 n 次试验中，事件 A 发生的次数。

当 n→∞n \to \inftyn→∞，频率（nAnn_A \over n​n​​n​A​​​​）稳定在某一个常数 P 附近。
且随着 n 的增大，摆动的幅度越来越小，此时称 A 为随机事件，称 p 为事件 A 发生的概率，记作：P(A)=pP(A) = pP(A)=p

1.5.2.2 数学定义

对于随即试验赋予一个实数P(A)P(A)P(A), 称为概率，满足已下条件

  1. 对于任何事件 AAA，有 P(A)≥0P(A) \ge 0P(A)≥0

  2. 对于必然事件 SSS，有 P(S)=1P(S) = 1P(S)=1

    也可以表示为 P(Ω)=1P(\Omega) = 1P(Ω)=1

  3. 可列可加性：对于两两不相容事件，则有 P(A1∪A2∪⋯∪An)=P(A1)+P(A2)+⋯+P(An)P(A_1 \cup A_2 \cup \cdots \cup A_n) = P(A_1) + P(A_2) + \cdots + P(A_n)P(A​1​​∪A​2​​∪⋯∪A​n​​)=P(A​1​​)+P(A​2​​)+⋯+P(A​n​​)

    也称为完全可加性

P(A)P(A)P(A) 表征事件 AAA 在一次试验中发生的可能性大小。

上述要点也是概率的性质。
注意，概率是趋近得到的，概率为 1 的事件不一定是必然事件。
同理，概率为 0 的事件不一定是必然事件。

1.8 独立性

若 AAA BBB 两事件，满足 P(A∣B)=P(A)P(B)P(A|B) = P(A)P(B)P(A∣B)=P(A)P(B), 即 P(A∣B)=P(B)P(A|B) = P(B)P(A∣B)=P(B)

  1. 若 A,BA,BA,B 独立，则 P(A∣B)=P(B)P(A|B) = P(B)P(A∣B)=P(B)

  2. 若 A,BA,BA,B 独立，则 A,B‾; A‾,B; A‾,B‾A,\overline B;\ \overline A, B;\ \overline A, \overline BA,​B​​​; ​A​​​,B; ​A​​​,​B​​​

    推广：若积事件的概率都等于事件概率的积。那么事件相互独立

1.8.2 推论

  1. 若 A1,A2,⋯AnA_1, A_2, \cdots A_nA​1​​,A​2​​,⋯A​n​​ 相互独立，那么其中的任意 k 个事件也相互独立
